Line Follower Robot

Tentang Projek

Line Follower Robot adalah robot berbasis Arduino yang dirancang untuk mengikuti jalur garis secara otomatis. Menggunakan sensor inframerah (IR), robot ini mendeteksi garis dan mengontrol motor menggunakan driver L298N atau DRV8833. Data pergerakan dapat ditampilkan pada LCD, LED, atau Serial Monitor untuk memberikan informasi navigasi secara langsung.

Selain itu, sistem ini juga memungkinkan integrasi dengan platform web untuk pemantauan jarak jauh. Dengan menggunakan modul WiFi seperti ESP8266 atau ESP32, atau koneksi Bluetooth, pengguna dapat memantau pergerakan robot secara remote tanpa perlu berada di lokasi yang sama.

Fitur utama:

  • Navigasi otomatis mengikuti jalur garis dengan presisi tinggi
  • Menggunakan sensor Infrared
  • Sistem kontrol motor berbasis driver motor L298N atau DRV8833
  • Kecepatan dan arah dapat disesuaikan sesuai jalur yang diikuti

Alat ini sangat cocok untuk berbagai aplikasi seperti sistem navigasi robot otonom, jalur produksi otomatis, atau proyek edukasi di bidang robotika dan kecerdasan buatan.
